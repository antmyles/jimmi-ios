# Oura account and ring requirement findings

Official Oura API support documentation states that the Oura API is used to access data generated by the Oura Ring. It also states that to use the Oura API, a person needs an Oura account and an API application.

The same article states that Gen3 and Oura Ring 4 users without active Oura Membership will no longer be able to access their data through the Oura API, and partner applications integrated with the Oura API will no longer have access to those users' data without active membership.

Practical implication for JIMMI: it may be possible to reach Oura account creation or developer app screens without having a ring, but useful API testing and production user data access depend on Oura users having ring-generated data, and for Gen3/Ring 4 users, active Oura Membership.

Primary official source: https://support.ouraring.com/hc/en-us/articles/4415266939155-The-Oura-API
